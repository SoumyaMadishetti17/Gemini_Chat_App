# Gemini AI Clone

A modern AI assistant web application inspired by Google Gemini, built using **React + Vite**.
This app allows users to ask questions and receive AI-generated responses using the **Google Gemini API**.

---

## 🚀 Features

* AI-powered question answering
* Gemini-style user interface
* Real-time response generation
* Sidebar layout similar to Gemini
* Loading animation while fetching responses
* Context-based state management
* Fast and optimized with Vite

---

## 🛠 Tech Stack

* React
* Vite
* JavaScript (ES6+)
* CSS
* Google Gemini API

## ⚙️ Installation & Setup

### 1. Clone the repository

```id="x1j8he"
git clone https://github.com/SoumyaMadishetti17/Gemini_Chat_App.git
```

### 2. Navigate to the project

```id="p6m4b1"
cd Gemini_Chat_App
```

### 3. Install dependencies

```id="q0v5zk"
npm install
```

### 4. Add Gemini API Key

Create a `.env` file in the root directory and add:

```id="m2v7rt"
VITE_GEMINI_API_KEY=your_api_key_here
```

⚠️ Keep your API key private. Do not upload `.env` to GitHub.

---

## ▶️ Run the Application

```id="b0l9jd"
npm run dev
```

Open in browser:

```id="u5v2x8"
http://localhost:5173
```
